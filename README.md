# ScriptCam 🎥

**ScriptCam** is a professional teleprompter and video recording application built with Flutter. It features a glassmorphic UI, studio-grade camera controls (Exposure, Zoom, Flash), and a draggable script overlay.

## ✨ Features

- **Smart Teleprompter:** Draggable overlay with auto-scroll, speed, and opacity controls.
- **Pro Camera:** FHD recording, manual exposure, zoom (1x/2x), and flash modes.
- **Modern UI:** Custom glassmorphism design (no default OS overlays).
- **Post-Processing:** Integrated video preview and editor.

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (3.0.0+)
- Android Studio / VS Code
